Overview

This sample post shows you two different ways to use SQL in an R Markdown document.

R packages

library(DBI)
library(RSQLite)

Create a SQLite database

If you don’t have access to an exisitng SQL database, use SQLite. SQLite is a local, serverless database application. SQLite is perfect for building small apps and practicing SQL.

# create a SQLite db
db <- dbConnect(RSQLite::SQLite(), dbname = "mydb.sqlite")

# write the iris dataset to a table named IRIS
dbWriteTable(conn = db, name = "IRIS", value = iris, overwrite = T)

Pure SQL code chunks

SELECT Species, COUNT(1) as CNT 
FROM IRIS
GROUP BY Species
Table 1: 3 records
Species CNT
setosa 50
versicolor 50
virginica 50

Embedding SQL in R code

DBI::dbGetQuery(
  conn = db,
  statement = "
  SELECT Species, COUNT(1) as CNT 
  FROM IRIS
  GROUP BY Species
  "
)
##      Species CNT
## 1     setosa  50
